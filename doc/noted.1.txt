NOTED(1)
======
:doctype: manpage

NAME
----
noted - Noted ELN Electronic Lab Notebook

SYNOPSIS
--------
*noted* ['NOTEBOOK']

DESCRIPTION
-----------
The *noted*(1) command starts the Noted ELN Electronic Lab Notebook
editor.  Without arguments it presents a splash screen from which a
notebook can be selected. With argument, it loads up the specified
notebook.

OPTIONS
-------
None.

BUGS
----
Noted ELN is fairly mature and used daily in our lab. Noted ELN has
never caused anyone to lose any data or notes. However, there may be
bugs. Please contact the author if one bites you.
